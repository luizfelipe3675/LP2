Fiz um programa em Java2D onde o usuário escolhe quais figuras quer desenhar e quantas deseja criar. Depois disso, o programa pede as informações necessárias de cada figura, como posição, tamanho, cor RGB, espessura, rotação e se a figura será preenchida ou não. Também é possível adicionar textos, permitindo criar vários tipos de desenhos básicos.tentei organizar o projeto dividindo o código em alguns pacotes para deixar tudo mais simples de entender e mais organizado, separando a parte das figuras, da interface e da execução do programa.Por enquanto o programa funciona pelo terminal, mas futuramente irei adicionar uma interface gráfica com botões e ferramentas para facilitar ainda mais a criação dos desenhos.
